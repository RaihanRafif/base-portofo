# Nitro Coffee @Ciranjang — Mature Landing Page (EN default + language toggle)

## What this is
Single-page landing page (HTML/CSS/JS) with niche positioning:
**"work-friendly specialty coffee pit stop"**.

Default language: **English**
Language toggle: **EN / ID** (saved in localStorage)

## Run
Open `index.html` directly or use a local server.

## Images
Place images in: `assets/images/`

This page references these filenames (JPG):
- `manual_2017_exterior_bluebrick.jpg`
- `manual_2017_interior_counter.jpg`
- `lifenesia_2024_drink_close.jpg`

If missing, it falls back to: `assets/images/placeholder.svg`.

### Getting reference images
Run `scripts/download_images.py` from your reference pack, then copy results into this folder.

## Hours note
Public sources vary, so this demo uses a pragmatic default (07:00–21:00). Confirm on Google Maps before publishing.

## Copyright note
Reference photos from the web are not included. Publish only photos you own or have permission to use.
